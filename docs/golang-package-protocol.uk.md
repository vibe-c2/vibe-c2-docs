# vibe-c2-golang-protocol

Репозиторій: `https://github.com/vibe-c2/vibe-c2-golang-protocol`

## Встановлення

```bash
go get github.com/vibe-c2/vibe-c2-golang-protocol@v0.1.0
```

## Призначення

- канонічні контракти повідомлень для комунікації між каналом та C2
- спільні константи (`inbound.agent_message`, `outbound.agent_message`, version)
- допоміжні функції валідації та типізовані помилки валідації

## Поточний стан

- випущена версія: `v0.1.0`
- доступний через екосистему Go-модулів (`proxy.golang.org`)
