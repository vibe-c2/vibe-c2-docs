# Записи архітектурних рішень (ADR)

Використовуйте ADR для фіксації важливих технічних рішень.

## Конвенція найменування

- `0001-short-title.md`
- `0002-short-title.md`

## Мінімальний шаблон

```md
# ADR-000X: Назва рішення

## Статус
Proposed | Accepted | Superseded

## Контекст
Яку проблему ми вирішуємо?

## Рішення
Що було вирішено?

## Наслідки
Позитивні, негативні та компроміси.
```
